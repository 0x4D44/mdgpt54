## Process

1. Ground-truth: `npm run check:types` (Ă—3 tsconfigs) + `vitest run`; record actual pass/fail. *(running)*
2. Fan out the 7 roles in parallel; each returns structured findings: `{id, severity, file:line,
   claim, evidence, recommendation, confidence}`. Severity = Critical / High / Medium / Low / Nit.
3. Devil's advocate runs against the *codebase and the other reviews' framing*.
4. Synthesize: merge, de-dup, reconcile contradictions, separate confirmed from speculative, fold in the
   devil's-advocate reframing. Sanity-check high-severity claims against the actual source before
   asserting them.
5. Write `wrk_docs/YYYY.MM.DD - CR - worldviewer comprehensive review.md`: exec summary, severity-ranked
   findings with evidence, thematic analysis, devil's-advocate section, prioritized recommendations.
